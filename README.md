# Amigo-Secreto :ninja:

## Código em python que fiz para realizar o sorteio da brincadeira amigo secreto.

- O método shuffle da biblioteca random é o responsável por realizar o "sorteio";
- Para abrir o browser utilizei a função open da biblioteca webbrowser;
- Por fim, a biblioteca pyautogui e a função sleep são os responsáveis pela robotização que envia as mensagens.
